# Frontend

The frontend is built with React and Vite.

## Run

```bash
cp .env.example .env
npm install
npm run dev
```

## Environment

Set the backend API URL in `.env`:

```bash
VITE_API_BASE_URL=http://localhost:8080
```

## Build

```bash
npm run build
```
